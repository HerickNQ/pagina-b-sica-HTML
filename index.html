História dos computadores

< inicio

A evolução dos computadores está intimamente ligada ao desenvolvimento da sociedade nos séculos XX e XXI, mas suas raízes remontam à antiguidade, com o ábaco, considerado o primeiro computador. Ao longo da história, diversas invenções, como a régua de cálculo de John Napier e a máquina de calcular automática de Pascal, contribuíram para o avanço da computação. Charles Babbage, no século XIX, criou a máquina analítica, sendo reconhecido como o "Pai da Informática". A evolução dos computadores é dividida em quatro gerações: a primeira (1951-1959) com válvulas eletrônicas, a segunda (1959-1965) com transistores, a terceira (1965-1975) com circuitos integrados e a quarta (1975 até hoje) com microprocessadores, que possibilitaram a miniaturização e a popularização dos computadores pessoais.

O que são clients?

< inicio

Clients é tudo aquilo que pode ser visto do lado do cliente com os seus navegadores, exemplos de linguagens de programação que temos do lado do cliente é: HTML, CSS e Javascript. E os profissionas responsáveis por cuidar dessa parte são chamados de desenvolvedores front-end.

O que são servers?

< inicio

Um server ou servidor é um programa que entrega a página solicitada em uma pesquisa virtual. Ele serve para que o usuário consiga a autorização para acessar uma página pelo seu navegador, enviando conteúdo e recursos, como páginas HTML, imagens, vídeos e arquivos, para os dispositivos dos usuários conectados à internet

Como é uma estrutura básica de um HTML?

< inicio

A estrutura básica de um HTML é composta por 3 tags essencias as tags HTML, HEAD e BODY. Lembrando sempre que todas elas acompanham os sinais < > para realizar a abertura da tag exemplo: <HTML>. E quase todas as tags devem ser fechadas com sinais </> Exemplo: </HTML>.

Entendendo a diferença entre listas ordenadas e não ordenadas

< inicio

A diferença é bem simples de ser identificada pelo browser(navegador) pois é bem fácil de diferenciar visivelmente a lista ordenada ela segue uma ordem tanto numérica como até em letras, já as listas não ordenadas elas tem sinais que identificam elas, que somente pode ser removido utilizando o CSS.

Já para identificar elas direto pelo HTML temos que cuidar a qual tag estamos utilizando para uma lista ordenada utilizamos a tag: <ol> seguido de suas linhas <li>. Enquanto nas listas não ordenadas utilizamos as tag: <ul> seguido de suas linhas <li>. Nunca esquecendo quando uma tag é aberta ela tem que ser fechada.
